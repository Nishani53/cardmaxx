# CardMaxx 💳

> Maximize every credit card purchase. Built by Nishani.

## Features
- 🔍 Find Best Card — pick a category, see all 20 cards ranked
- 💳 My Wallet — save your own cards for personalized results
- 🧮 Savings Calculator — see yearly rewards based on your spending
- ⚡ Deals & Offers — limited time promotions
- 🛒 Portal Stacking — stack portals on top of cards
- 📖 Guide — tips to maximize rewards

## Tech Stack
- React 18
- Pure CSS (no UI libraries)
- PWA (installable on phone)
